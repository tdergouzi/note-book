### 光标操作

| 命令                | 功能                          |
| ------------------- | ----------------------------- |
| shift + ⬆️⬇️⬅️➡️        | 上下左右选中                  |
| option + ⬅️➡️         | 以单词为单位移动              |
| cmd + d             | 选中光标所在单词              |
| ctrl + g            | 跳转指定行                    |
| option + cmd + ⬆️⬇️⬅️➡️ | 光标位置选择                  |
| option + shift + ⬆️⬇️ | 复制当前行内容至下一行/上一行 |
| ctrl + -            | 跳转至上次代码位置            |
|                     |                               |
